INSTALLATION
============

Prerequisites
Please install following extra packages:
   - libboost_system
   - libboost_regex

In debian with:
	sudo apt-get install libboost-system-dev libboost-regex-dev

Compile
   cd dnsitsme
   make

Create user "dnsitsme"

  sudo adduser dnsitsme
  
  
USAGE
=====

   sudo ./dnsitsme <ip-address>
